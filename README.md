# CyberHammer

A brave try for Channeling AI. Pure AI chain, based on [CyberResearch](https://github.com/QianZe-HAO/CyberResearch)

## Set Up Environment

### 1. Install `uv` (Python package manager)

By opening this project in Dev Container, it will automatically install `uv` for you. If you're not using Dev Container, you can install it manually.

### 2. Install Playwright dependencies
```bash
uv run playwright install-deps
uv run playwright install chromium
```

### 3. Install Python dependencies
```bash
uv sync
```
